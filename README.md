# QR-code-generator
A simple QR code generator

To use:
1. Inside the desired directory, open a terminal and run:
   $ pip install qrcode[pil]
   to install qrcode library. [pil] is optional to install additional dependencies; then wait for the the library to be downloaded.

2. Download or copy the code and run it to have access to the generator.

ps: keep in mind that the generated images will be saved next to the script.
